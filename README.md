### OneMessage

Installation:

    1. Git clone https://github.com/blackant-solutions/gio-whatsapp.git
    2. cd gio-whatsapp
    3. compser install
    4. npm install
    5. Update DB credentials on .env file
    6. php artisan migrate
    7. php artisan db:seed --class=FieldSeeder
    8. php artisan db:seed --class=ServiceSeeder
